# Hidden Band Info - 利用ガイド

Hidden Band Info は、Pixelシリーズ等の Android 端末で詳細なモバイル通信バンド情報をリアルタイムでモニタリング・記録するためのツールです。
---

## 🚀 インストール手順

本アプリはシステムの隠し API を利用するため、**[Shizuku](https://shizuku.rikka.app/)** アプリとの連携が必須です。

1.  **Shizuku のインストールと起動**
    - Google Play ストア等から [Shizuku](https://play.google.com/store/apps/details?id=moe.shizuku.privileged.api) をインストールします。
    - 「ワイヤレスデバッグ」等を使用して Shizuku サービスを開始してください。
    
2.  **本アプリのインストール**
    - 配布された APK ファイルをインストールします。
      
3.  **権限の許可**
    - アプリ起動時に表示されるダイアログに従い、`位置情報` や `電話の状態` の権限を許可してください。
    - Shizuku の利用確認ダイアログが表示されたら「常に許可」を選択してください。
    （またはShizukuアプリ上で権限の付与を行ってください）

## 📱 各画面の説明と使い方

### 1. メイン画面 (Home)
アプリを起動すると最初に表示される画面です。

- **バンド情報表示**: 接続中のスロットごとに、キャリア名（PLMN）、バンド番号、周波数（ARFCN）、PCI、電波強度（RSRP/RSRQ/SINR）を表示します。
- **タイムライン・シークバー (画面上部)**: 記録中のデータの進捗を表示します。スライダーを動かすことで、過去の接続状況をリアルタイムに振り返ることができます。
- **アクションボタン**:
    - **PIP**: 小窓表示（ピクチャー・イン・ピクチャー）に切り替えます。
    - **録画 (REC)**: ローカルデータベースへの記録を開始/停止します。
    - **エクスポート (Export)**: 記録したデータを CSV 形式で保存します。
    - **設定 (Settings)**: アプリの動作設定を開きます。

### 2. CSV 再生画面 (CSV Playback)
エクスポートした CSV ファイルや、過去のログを読み込んで閲覧する画面です。

- **テーブル表示**: 記録されたデータをリスト形式で表示します。
- **カラム選択**: 表示したい項目（CellID, PLMN, SINR など）を自由に選択して絞り込むことができます。
- **シークバー**: 履歴画面と同様に、ログ内の特定時刻へジャンプできます。

### 3. PIP モード
他のアプリ（地図やブラウザ等）を使用しながら、画面の隅にバンド情報を常時表示できます。

- **カスタマイズ**: 設定画面から、PIP 内のフォントサイズや情報の更新間隔を調整可能です。

## 🛠 技術的な制限事項
- **GLAXY端末での動作不備**: Galaxy系で **NRの情報取得が正しく出来ない事」** を確認しています。目下デバッグ中（制約事項になる可能性）
- **バンド番号の補完**: OS からバンド番号が直接取得できない場合、ARFCN（周波数）から推定したバンド番号を表示します。
- **不正確な表示**: 常に取得が出来ない情報があったり不正確な情報が表示される事があります。（特にDualSIM利用時）
- **PIP画面の問題**: アクティブなSIMのみ表示は動作しません。他の項目も一部未実装です。
